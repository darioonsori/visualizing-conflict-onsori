Visualizing Conflict and Human Suffering through Data

Comparing categories with open data (OWID / UCDP)

Introduction

This website explores how data visualization can represent global conflicts and humanitarian issues. Using open data from Our World in Data and the Uppsala Conflict Data Program (UCDP), we compare categories through multiple visual encodings to highlight different perspectives on the same underlying facts.

ℹ️

Note: “Extrasystemic” conflicts (wars fought outside a state’s own territory, e.g. colonial wars) are included for historical completeness but are no longer observed in recent decades, resulting in zero values in modern datasets.

Distributions: exploring variability across countries

This section explores how the number of conflict-related deaths varies across countries and conflict types in the snapshot year 2023. The same UCDP dataset is used but with a focus on statistical distributions.

Histogram — total conflict-related deaths per country (2023)

Each bar shows how many countries fall within a range of total conflict deaths. Aggregates and regions are excluded.

Violin plot — distribution by conflict type (2023)

For each conflict type, the violin shape shows the density of country-level death counts. The white dot indicates the median; the black line spans the interquartile range.

Boxplot — country-level distribution by conflict type (2023)

Boxes represent the interquartile range (Q1–Q3), the line marks the median, and whiskers extend to non-outlier min/max values.

Spatial patterns

This section explores how conflict-related deaths are distributed across space. All maps use the same UCDP dataset, focusing on country-level data for the snapshot year 2023.

Choropleth — conflict-related deaths per country (2023)

Each country is coloured according to its total number of conflict-related deaths recorded in 2023. Darker shades indicate higher values; countries with no data are shown in light grey.

Countries appearing in light grey are those for which the UCDP dataset does not report any conflict-related deaths in 2023. This does not necessarily mean that data are missing, but rather that no conflict events reached UCDP’s reporting threshold in that year.

World choropleth map based on UCDP country-level totals (combatants and civilians) for 2023.

Proportional symbol map — deaths represented as circles (2023)

This map encodes the same 2023 country totals as the choropleth, but uses proportional circles instead of colour. Circle area is proportional to the number of conflict-related deaths, following standard cartographic guidelines for proportional symbol maps.

Proportional symbol map based on UCDP 2023 country-level totals (combatants and civilians). Circle area scales with the number of deaths.
