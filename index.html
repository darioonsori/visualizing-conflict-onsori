Visualizing Conflict and Human Suffering through Data

Comparing categories with open data (OWID / UCDP)

Introduction

This website explores how data visualization can represent global conflicts and humanitarian issues. Using open data from Our World in Data and the Uppsala Conflict Data Program, we compare categories with multiple encodings to surface different perspectives on the same underlying facts.

Distributions: comparing categories with density and quartiles

The charts below use the same UCDP/OWID dataset and focus on distributions across countries for the snapshot year 2023. Aggregates (e.g., “World”, continents) are excluded via ISO3 filtering.

Histogram of conflict-related deaths per country (2023)

Each bar counts the number of countries falling in the bin of total deaths during ongoing conflicts (UCDP). ISO3 filter removes regional aggregates.

Violin plot — distribution by conflict type (2023)

For each conflict type, the violin encodes the kernel density of country-level deaths. The white dot marks the median; the line spans the interquartile range.

Boxplot — distribution by conflict type (2023)

Box shows Q1–Q3 with the median; whiskers extend to non-outlier min/max. Values are country-level totals per conflict type; aggregates excluded.
